# Historia Nagrody Nobla

Projekt wykonany na platformie DataCamp. Poświęcony analizie historycznych danych o laureatach Nagrody Nobla w latach 1901–2023. 
Wykorzystując biblioteki Pandas, Seaborn oraz Matplotlib, zbadałem trendy demograficzne, geograficzne oraz płeć laureatów na przestrzeni dekad.


## Wnioski

Na podstawie analizy udało się ustalić następujące fakty:
- Najwięcej nagród trafiło do osób urodzonych w Stanach Zjednoczonych, a najczęściej nagradzaną płcią są mężczyźni.
- Dekadą, w której amerykańscy laureaci stanowili największy odsetek wszystkich zwycięzców, były lata 2000.
- Pierwszą kobietą nagrodzoną Noblem była Maria Skłodowska-Curie w dziedzinie fizyki.
- Analiza potwierdziła, że istnieją wybitne jednostki i organizacje, które otrzymały nagrodę więcej niż raz, m.in. Międzynarodowy Komitet Czerwonego Krzyża, Linus Pauling, Marie Curie czy Frederick Sanger.
